### Gateway Auth Hardening (v2026.3.31 Breaking)
`trusted-proxy` now rejects mixed shared-token configs, and local-direct fallback requires the configured token instead of implicitly authenticating same-host callers. **If we rely on localhost auto-auth, this could break things after update.**

### Plugin/Skill Install Security (v2026.3.31 Breaking)
Dangerous-code findings now fail closed by default. Plugin installs that previously succeeded may need `--dangerously-force-unsafe-install`. Watch for this on next skill/plugin install.

### Node Security Tightening (v2026.3.31 Breaking)
- Node commands stay disabled until pairing is approved (not just paired)
- Node-originated runs use reduced trusted surface
- Impact: If we use any node-triggered flows, they may need adjustment.

## 🟡 Worth Exploring

### /tasks Command (v2026.4.1)
New `/tasks` chat command shows a background task board for the current session. Useful for monitoring cron jobs and subagent work.

### macOS Voice Wake (v2026.4.1)
Voice Wake trigger for Talk Mode on macOS. Could be interesting for hands-free interaction.

### Cron Tool Allowlists (v2026.4.1)
`openclaw cron --tools` lets you restrict which tools a cron job can use. Good for security hygiene on our automated jobs.

### agents.defaults.params (v2026.4.1)
Global default provider parameters — could simplify config if we want consistent params across agents.

### Matrix Streaming (v2026.3.31)
Draft streaming for Matrix — partial replies update the same message instead of sending chunks. Major UX improvement for our Matrix channel.

### Matrix History Context (v2026.3.31)
`channels.matrix.historyLimit` adds room history context for group triggers with per-agent watermarks. Useful for our Matrix group chats.

### Matrix Proxy Config (v2026.3.31)
`channels.matrix.proxy` for routing Matrix traffic through HTTP(S) proxy.

### Matrix Thread Overrides (v2026.3.31)
Per-DM `threadReplies` overrides for Matrix thread session isolation.

### Background Tasks Overhaul (v2026.3.31)
Unified task control plane (ACP, subagent, cron, background CLI) under SQLite-backed ledger. `openclaw flows list|show|cancel` for task flow management.

### Idle Stream Timeout (v2026.3.31)
Configurable timeout for stalled model streams — no more hanging on stuck providers.

### SearXNG Web Search (v2026.4.1)
Bundled SearXNG provider for self-hosted web search. Alternative to Brave.
