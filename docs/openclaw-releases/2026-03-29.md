## 🟡 Worth Exploring: New Features in 2026.3.28

### Plugin tool approval hooks (#55339)
Plugins can now pause tool execution and prompt for user approval via Telegram buttons, Discord interactions, or `/approve`. Useful if we want finer-grained control over what tools can do in group chats.

### xAI / Grok web search (#56048)
Bundled Grok provider moved to Responses API with native `x_search`. Auto-enables from web-search config — no manual plugin toggles needed. New onboarding option for x_search setup.

### ACP bind-to-current-chat
`/acp spawn codex --bind here` now works in Discord, BlueBubbles, and iMessage — turns current chat into a Codex workspace without child threads.

### Matrix TTS voice bubbles (#37080)
Auto-TTS replies now send as native Matrix voice messages instead of generic audio attachments. Directly relevant to our Matrix setup.

### Slack upload-file action + Teams/Google Chat file sends
Explicit file upload action for Slack, plus unified file-send support for Teams and Google Chat.

### OpenAI apply_patch enabled by default
Now on by default for OpenAI/Codex models with sandbox-aligned write permissions.

### Gemini CLI as bundled backend
Gemini CLI joins Claude CLI and Codex CLI as an auto-loaded plugin backend — no manual `plugins.allow` needed.

### `openclaw config schema` command (#54523)
Print the generated JSON schema for openclaw.json — useful for config validation.
